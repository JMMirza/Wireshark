# Wireshark
Hey there its a web application just like the wireshark and contain two folders:
    1. Frontend (React js)
    2. Backend (Node js)
To start backend there are two command that you have to run:
    1. npm i
    2. node index.js

To start frontend you have to write two commands:
  For yarn:
    1. yarn
    2. yarn start
  For npm:
    1. npm i
    2. npm start
